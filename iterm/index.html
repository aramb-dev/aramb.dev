Installation Guide

Prerequisites

  1. Download and install iTerm2 if you haven't already
  2. Download the configuration files from the section above

Method 1: Import Color Preset

  1. Open iTerm2
  2. Go to iTerm2 > Preferences > Profiles > Colors
  3. Click on Color Presets... dropdown
  4. Select Import...
  5. Navigate to and select the downloaded aramb-dev.json file
  6. Click on Color Presets... again and select aramb-dev

Method 2: Import Key Bindings

  1. Open iTerm2
  2. Go to iTerm2 > Preferences > Keys > Key Bindings
  3. Click on Presets... dropdown
  4. Select Import...
  5. Navigate to and select the downloaded key-bindings.itermkeymap file

Method 3: Complete Configuration

  1. Quit iTerm2 completely
  2. Open Finder and press ⌘+Shift+G
  3. Enter ~/Library/Preferences/ and click Go
  4. Make a backup of your current com.googlecode.iterm2.plist file if it exists
  5. Copy the downloaded com.googlecode.iterm2.plist to this directory
  6. Restart iTerm2

Note: This method will replace your current iTerm2 settings. Make sure to back up your existing configuration first.

Advanced Configuration

Customizing the Profile

You can further personalize the profile by modifying:

  • Font type and size (I use Monaco 12pt)
  • Background transparency level (currently set to ~10%)
  • Terminal window style and dimensions
  • Status bar components and layout

Access these settings through iTerm2 > Preferences > Profiles after importing the configuration.

Shell Integration

My configuration includes iTerm2's shell integration features. To enable these fully:

  1. Open iTerm2
  2. Go to iTerm2 > Install Shell Integration
  3. Follow the on-screen instructions

This enables features like command history tracking, better navigation, and more.
